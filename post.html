- Aplicação criada para treinar os conceitos de fetch.

- Trata-se de uma simulação de um blog.

- O script realiza a requisição e envio de dados para uma API, demonstrando os resultados na página, através dos métodos:

- GET: demonstra todos as postagens da API no index.html e individualmente no post.html

- POST: cada postagem possui uma seção de comentários e o usuário pode publicar um novo, que aparecerá na lista, após o envio.

- A API utilizada é "mockada", portanto somente simula as rotas, sem armazená-las em banco de dados.
